# proyectoCafe Conocimiento

Sistema de gestión de café: Django (web), FastAPI (API), PostgreSQL y ontología RDF en Apache Jena Fuseki.

El código está en la carpeta `Zunga/`.

## Estructura

- `Zunga/cafeAPI/` — API FastAPI (puerto 8001), Fuseki y PostgreSQL
- `Zunga/proyectoCafe/proyectoCafe/` — Django (puerto 8000)
- `Zunga/COMANDOS_INICIO.txt` — Cómo levantar Fuseki, API y Django

## Requisitos

- Python 3.11+
- PostgreSQL (`bdcafe`)
- Apache Jena Fuseki (dataset `cafe`, puerto 3030)

Ver `Zunga/COMANDOS_INICIO.txt`.
